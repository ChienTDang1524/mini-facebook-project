# Mini Facebook Project

## Giới thiệu
**Mini Facebook** là một ứng dụng mạng xã hội thu nhỏ với đầy đủ các tính năng cơ bản: đăng bài, like, comment, quản lý profile. Dự án sử dụng **Vue 3** cho frontend và **Node.js/Express** cho backend.

## Tính năng chính
✅ **Đăng ký/Đăng nhập** với JWT authentication  
✅ **Đăng bài** với text + ảnh + video  
✅ **Like/Unlike** bài viết  
✅ **Bình luận** và xóa bình luận  
✅ **Chỉnh sửa profile** + đổi avatar  
✅ **Upload đa phương tiện** (ảnh, video)  
✅ **Giao diện responsive** với Bootstrap 5
